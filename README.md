
 

# 🏗️ Data Warehouse & Analytics Project

This project demonstrates the end-to-end process of building a modern data warehouse using SQL, followed by preparing data for analytics and reporting.

The goal was to understand how raw data is transformed into structured, business-ready insights.

---

## 📊 Data Architecture

<img width="1010" height="722" alt="image" src="https://github.com/user-attachments/assets/d5767538-841f-4e54-a041-d1fcf69791d1" /># SQL-Data-Warehouse-Project-



This project follows the **Medallion Architecture**:

- **Bronze Layer**  
  Raw data ingestion from CSV files into SQL Server without modifications.

- **Silver Layer**  
  Data cleaning, transformation, and standardization to make data usable.

- **Gold Layer**  
  Business-ready data modeled using a **star schema** for analytical queries.

---

## ⚙️ Project Workflow

1. **Data Ingestion**  
   Imported ERP and CRM datasets (CSV) into SQL Server.

2. **Data Cleaning & Transformation**  
   - Handled null values  
   - Standardized formats  
   - Removed inconsistencies  

3. **Data Modeling**  
   - Created fact and dimension tables  
   - Optimized schema for analytics  

4. **Analytics**  
   - Built SQL queries to analyze:
     - Customer behavior  
     - Product performance  
     - Sales trends
